Service Limits, Quotas, and API Rate Limits FAQ for the AI Knowledge Assistant

This article lists the document, file-size, query, and API limits for each plan, plus how deletion and data retention work. Limits are enforced per workspace, and the daily counters reset at 00:00 UTC.

How many documents can I store on each plan?
The Free plan stores up to 50 documents. The Pro plan stores up to 5,000 documents. The Enterprise plan has no fixed document cap and is limited only by the contracted storage volume. Counts include every uploaded file but not the chunks created from it, so a single large file still counts as one document.

What is the maximum file size for a single upload?
A single file may be at most 20 MB on Free and Pro plans, and up to 100 MB on Enterprise. Supported formats are PDF, Markdown, plain text, and Word documents. Files larger than the limit must be split before uploading, and password-protected PDFs must be unlocked first because the indexer cannot read them.

How many search queries can I run per day?
Free allows 100 queries per day, Pro allows 5,000 queries per day, and Enterprise is unmetered. The daily counter resets at 00:00 UTC. When the limit is reached, search returns a "quota exceeded" message until the reset, and Admins receive an email at 80 percent of the daily quota so they can upgrade before users are blocked.

What is the rate limit for the API?
The API allows 60 requests per minute on Pro and 600 requests per minute on Enterprise; the API is not available on the Free plan. Exceeding the limit returns HTTP 429 with a Retry-After header. Clients should back off and retry after the number of seconds given in that header rather than retrying immediately, which only extends the throttling.

Is there a limit on embeddings or total storage?
Each document is split into chunks and embedded; embeddings count toward the contracted storage volume on Enterprise but are not separately capped on Free or Pro. If a workspace approaches its storage volume, new uploads are blocked while existing search keeps working, so you can delete old documents to free space.

How long are deleted documents retained?
Deleted documents move to a trash area and are permanently removed after 30 days. During those 30 days an Admin can restore them from Settings > Documents > Trash. After 30 days the document text and its embeddings are deleted and cannot be recovered, which also frees the storage they used.

Do limits change immediately when I upgrade?
Yes. Higher document, query, and API limits apply the moment an upgrade is confirmed, even mid-cycle. Downgrades apply at the next billing cycle, and if your current usage exceeds the lower plan's limits you must reduce usage before the downgrade can complete.
